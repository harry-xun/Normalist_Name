The Normalist Name.

What is this Data?

This website runs off a dataset that contains names given to babies born in the United States of America. You can find the data on the Social Security Website. (Last accessed Feb 29 2024)

This information in the dataset is sourced from The Social Security Administration(SSA) databases as of March 2023. It covers names from 1880 - 2022, In the states and territories of the USA. No modifications were done to the available dataset, outside of formatting.

The Dataset contains the amount of occurrences of the name, assigned to both males and females, per year. This allows us to examine trends of name popularity, usage for males and females, and get period specific names. Because we are using the full database here (as opposed to the SSA's top 1000) we have a wider and more diverse set of names, and more tools for analysis over time. We hope that this tool will be useful for getting name ideas and learning about usage history of names.

Licensing

Per data.gov, this dataset is licensed under Creative Commons CCZero, meaning that the data is intended for public access and use.

Limitations

This data does not represent all names, only ~70% of the population, as names with less than 5 occurrences in a year are hidden for privacy reasons.
